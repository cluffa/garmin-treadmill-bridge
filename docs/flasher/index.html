flash the bridge

Plug your ESP32 into USB, pick your board, hit flash. The firmware is pulled live from the latest nightly release. No IDF, no esptool, no cables to the terminal — just the browser.

Heltec WiFi LoRa 32 v3

ESP32-S3 · OLED + battery

The board with the little screen and battery management.

This browser can't talk to serial ports. Use desktop Chrome or Edge. Flashing needs a secure (https) page — open this over https.

XIAO ESP32-C6

ESP32-C6 · compact, USB-C

The small one. Primary board for the bridge.

This browser can't talk to serial ports. Use desktop Chrome or Edge. Flashing needs a secure (https) page — open this over https.

Got the XIAO nRF52840 ANT+ variant? That one flashes over SWD, not USB serial — it can't be done from the browser. See CLAUDE.md for the build.
